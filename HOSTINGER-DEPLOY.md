### Si problème de connexion SSH :
```bash

```

### Si build échoue :
- Vérifiez les **GitHub Secrets**
- Contrôlez la **version Node.js** (18+)
- Videz le **cache npm**

### Si page blanche :
- Videz le **cache navigateur** (Ctrl+F5)
- Vérifiez la **console** (F12)
- Contrôlez les **permissions fichiers**
